# reclaimshield-site

Production site for **reclaimshield.io** — platform enforcement response service operated as a registered fictitious name of Sentinel Risk Group, LLC (Florida).

## Structure

- `/` — hub / platform selector
- `/meta/` — ReclaimShield for Meta (Instagram, Facebook, Threads)
- `/amazon/` — ReclaimShield for Amazon (Seller Plan of Action)
- `/privacy/` — Privacy Policy
- `/terms/` — Terms of Service
- `CNAME` — GitHub Pages custom domain (reclaimshield.io)

## Deploy

GitHub Pages is enabled on `main`. Pushes to `main` auto-deploy (~30s build time).

## Brand system

- Typography: Inter (body), Space Grotesk (display)
- Palette: `#0A0A0F` bg / `#7C3AED` purple / `#2DD4BF` teal
- Amazon subpage adds accent: `#FF9900` marigold + `#146EB4` deep blue (Amazon brand adjacency, not affiliation)
- Trust layer constant across all pages: footer disclaimer, "not a law firm" language, Sentinel Risk Group LLC attribution

## Not a law firm

All pages carry the canonical disclaimer. ReclaimShield does not provide legal advice, does not create an attorney-client relationship, and is not affiliated with Meta Platforms, Inc. or Amazon.com, Inc.
